Traduction des armures:

Chainmail = cotte de maille

Knight's armor = armure de chevalier

Hunter's jacket = veste de chasseur

Axe des abscisses (horizontal) = % de critiques DE VOTRE ADVERSAIRE que vous souhaitez évaluer (évaluez un crit moyen de vos adversaires)

Axe des ordonnées (vertical) = diminution en % des dégâts par seconde en fonction de l'armure

Données à saisir dans le graphique:

ligne 7 => les % de critiques de votre adversaire (ou le % de critiques que vous souhaitez évaluer)

ligne 9 à15 => saisi des talents unités ou héro de son deck qui influent les DÉGÂTS CRITIQUES  (exemple: talent 9G de pistolero qui augmente de 20% les dégâts crit / Statue 13G augmente 15% crit ou Zeus 30 qui augmente de 5% les dégâts crit)

Il faut saisir sur chaque ligne les différentes augmentations de crit de son deck

ligne 17 => saisi des % de chances de coup critique.

De base, tous les joueurs ont 5% de chance de crit. A cela, il faut ajouter les augmentations des talents de cartes et de héros (exemple: talent 13G Danse Lame qui augmente de 5% / Statue T7 L15 augmente de 8% chance de crit / Gadget 30 augmente 4% chance de crit)

Ligne 19 => définir si les dégâts sont effectués sur un anti-crit ou sur un boss (0 = boss / 1 = anti crit)

Ligne 20 => paramètres de votre armure de chavlier

Ligne 22 => définir le niveau de votre armure de chevalier

Ligne 24 => nombre d'unité de faction qui affectent votre armure

Ligne 26 (bonus set) => Laisser ce paramètre à 0 !! De base fait pour les 10% bonus de l'ancienne MAJ, le fichier doit être refait avec les nouvelles données

Ligne 27 => laisser ce paramètre à 0

Ligne 29 => paramètre de la cotte de maille

Saisir comme pour l'armure de chevalier

Ligne 38 => paramètres de la veste de chasseur

Saisir comme pour l'armure de chevalier

Vous obtenez ainsi un graphique selon vos données, qui vous permet de connaitre la meilleure armure à utiliser pour votre deck défini.
